# vue3-skeleton

```sh
$ npm install @keeveeg/vue3-skeleton
```

```javascript
import '@keeveeg/vue3-skeleton/style.css'
import Skeleton from '@keeveeg/vue3-skeleton'
app.use(Skeleton)
```

```html
<Skeleton :ready="ready" width="200" height="1rem">
  <div>I am ready. 1234567890</div>
</Skeleton>
```
